## 📋 О проекте

АЙДА — это современная премиальная доска объявлений с поддержкой TikTok-ленты, карты, чатов в реальном времени, аукционов, лотереи и бонусной системы.

### ✨ Основные возможности

- 🎬 **TikTok-лента** — вертикальная прокрутка с видео и жестами
- 🗺️ **Карта объявлений** — маркеры с кластеризацией, поиск по радиусу
- 💬 **Чаты в реальном времени** — WebSocket, автоответчик, быстрые ответы
- 🎰 **Лотерея** — еженедельные розыгрыши с призовым фондом
- 🎁 **Бонусная система** — ежедневные бонусы, рефералы, достижения
- 🔨 **Аукцион** — торги в реальном времени
- 📱 **PWA** — установка на телефон, офлайн-режим
- 🔐 **Безопасность** — JWT, CSRF, rate limiting, защита от XSS/SQL

## 🛠️ Технологический стек

### Бэкенд
- **Runtime**: Node.js 18+
- **Framework**: Express.js
- **База данных**: PostgreSQL 15
- **Кеш**: Redis 7
- **WebSocket**: Socket.IO
- **Очереди**: Bull
- **Аутентификация**: JWT, Passport.js (Google, Яндекс, VK, Telegram)

### Фронтенд
- **HTML5/CSS3/JavaScript ES2022**
- **Карты**: Leaflet + OpenStreetMap
- **Графики**: Chart.js
- **Анимации**: CSS3

### DevOps
- **Контейнеризация**: Docker, Docker Compose
- **Веб-сервер**: Nginx
- **Мониторинг**: PM2
- **CI/CD**: GitHub Actions
